# Vivaldi

Version: 2.12.1862.3 (正式版本) （64 位）

![screenshot](./screenshot.png)

## Install

```sh
brew cask install vivaldi
```

## Shortcut

* Cmd+E: 打开快捷命令
* Cmd+L: 聚焦地址栏
* Cmd+Option+N: 打开笔记面板
* Ctrl+Tab: 循环标签

## Problems

* 隐藏扩展后，扩展的快捷键不生效：通过右键使用扩展
